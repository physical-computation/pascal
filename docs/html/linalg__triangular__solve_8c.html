Solve a linear system of equations with a triangular matrix.

Solve a linear system of equations Ax = y for x, where A is a triangular matrix. The solver runs on the last two dimensions of the tensors. The other dimensions are broadcasted (in the NumPy sense).
Parameters
aThe triangular matrix A
yThe vector y
lowerWhether the matrix is lower triangular (true) or upper triangular (false)
Returns
The solution x
Exceptions
IfTENSOR_USE_ASSERT is set a fatal error is raised if the matrix a is not square (in the last two dimensions).
IfTENSOR_USE_ASSERT is set a fatal error is raised if the shapes aren't compatible for broadcasting.
Note
This function does not check if the matrix is triangular.
